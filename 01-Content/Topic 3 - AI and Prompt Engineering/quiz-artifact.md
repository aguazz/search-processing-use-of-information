# Quiz Artifact — recipe

The self-scoring quiz you build in **Part 4** is an **Artifact**, generated live from an
**AIM** prompt. Here's the exact recipe: paste it into a chat *inside your Project* (so it
draws on the sources you've already uploaded) and adapt the bracketed parts.

## The prompt (AIM)

- **Actor** — You're a quiz-maker for university courses.
- **Input** — [the syllabus / notes already in this Project].
- **Mission** — a 5-question self-scoring practice quiz about [topic], built as an
  interactive **Artifact** (not a text answer), so I can test myself and see my score.

## Why it's shaped this way

- It's pure **AIM** — Actor + Input + Mission. "As an Artifact" belongs *inside* the
  Mission (it's the shape of what you want), not as a separate rule.
- Asking **explicitly** for an Artifact is what turns a plain list of questions in the
  chat into something interactive and reusable you can keep in the Project.

## Make it yours

- Swap "5-question quiz" for flashcards, a diagram, or a longer mock exam.
- Point the Mission at one topic, one week, or the whole course.
- Ask it to show the source (page or section) behind each answer, so the Artifact can be
  checked against your own material.
