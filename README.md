# Search, Processing and Use of Information

Teaching materials for the first-year CUNEF Universidad course *Search,
Processing and Use of Information* — an introduction to finding, handling, and
reasoning about data, plus a hands-on Excel track and a module on using AI well.

Shared openly for students to keep and for other educators to reuse or adapt.

## What's here

Everything lives under [`01-Content/`](01-Content/): the six theory topics, three
Excel modules, and the AI module. Slides are provided as both `.pptx` (editable)
and `.pdf` (ready to read); exercise data ships as `.xlsx`.

| | |
|---|---|
| **Topic 1 — Introduction** | Data → information → intelligence; what the course is about |
| **Topic 2 — Data Collection** | Where data comes from; surveys vs. behavioural data, sampling and bias |
| **Topic 3 — AI & Prompt Engineering** | Using Claude well: prompting, grounding, Projects, Skills, agents — with a keepable toolkit (see below) |
| **Topic 4 — Building Data Teams** | Roles on a data team and how they fit together |
| **Topic 5 — Ethical Uses of Data** | Privacy, consent, and responsible use |
| **Topic 6 — Types of Data Analytics** | Descriptive → diagnostic → predictive → prescriptive |
| **Topic 7 — EDA & Descriptive Analytics** | Exploring and summarising data |
| **Excel Modules 1–3** | Introduction · Formulas & Functions · Pivot Tables |

The [syllabus](Syllabus-Search_Processing_and_Use_of_Information.pdf) at the root
is the source of truth for scope and sequencing.

## Topic 3: the AI toolkit

[`Topic 3 - AI and Prompt Engineering/`](01-Content/Topic%203%20-%20AI%20and%20Prompt%20Engineering/)
is a self-contained mini-course on using Claude, with tools you keep past the module:

- **The slide deck** — six parts, from "what is a prompt" to building your own agent.
- **`prompt-cheat-sheet.html`** — one-page reference (ICC / AIM, the prompt checklist, techniques, asking for an Artifact). Open in a browser; print to PDF for a paper copy.
- **`prompt-doctor.html`** — paste a prompt, see whether ICC or AIM fits, rebuild it field by field.
- **`skills/`** — four example Claude **Skills** (`SKILL.md` files) to read, adapt, or upload.
- **`study-buddy/`** — the **Study Buddy agent** spec: quizzes you from your own course notes, tutors you when you're wrong, and searches the web for gaps.

## What's *not* here

No exams, exam solutions, question banks, grading rubrics, student submissions,
grades, or rosters — those stay in a separate private repository. Nothing in
this repo is graded or student-identifying.
